# Avion Furniture

Простой лендинг мебельного бренда

## Описание

Статическая HTML-страница с основными секциями интернет-магазина мебели: навигация, баннер, товары, преимущества бренда, форма подписки и футер.

## Используемые технологии

* HTML5
* Семантическая вёрстка
* Изображения и иконки

## Особенности

* Чистая структура HTML
* Использование `header`, `main`, `section`, `article`, `footer`
* Форма подписки с полем email
* Подготовлено для дальнейшего подключения CSS и JavaScript

## Структура проекта
* `index.html` - главная страница
* `assets/img` - изображения
* `assets/icons` - иконки
